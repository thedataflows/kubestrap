# kubestrap

> Note: this software is an alpha, work in progress.

Automate preparation of Kubernetes nodes and Kubernetes installation itself in self-hosted environments.

## Overview

CLI acting as a wrapper over predefined binaries, to ease bootstraping and maintenance of Kubernetes clusters and the apps deployed on them, in a GitOps manner.
